# Unity RenderGraph Blur
이 자료는 유니티 6의 Render Graph 방식으로 Blur를 처리하는 예제입니다.  
블러 방식은 다운 샘플링과 상하좌우로 이동하여 얻는 빠르지만 퀄리티 괜찮은 방식을 사용합니다. (그저.. 가벼움)  

Sprite 블러 방식과 UI 블러 방식 두가지를 준비해봤으니 관심있는 브랜치로 이동하여 프로젝트를 뒤적뒤적 하십시오.
